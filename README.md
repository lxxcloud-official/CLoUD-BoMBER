# CLoUD BoMBER💣


## قابلیت ها ♦

-   امکان استفاده از پروکسی
-   امکان استفاده از threads 
-  دیزاین زیبا 
-   سریع و پر سرعت 

## 📃 پیش نیاز ها
- [Python3.11](https://www.python.org/downloads/)
- requests
- PyStyle
## نصب و راه اندازی


```bash
  git clone https://github.com/lxxcloud-official/CLoUD-BoMBER
```


```bash
  cd CLoUD-BoMBER
```


```bash
  pip install -r requirements.txt
```


```bash
  python main.py
```

## استفاده از پروکسی: 
در صورتی که تمایل به استفاده از پروکسی دارید میتونید پروکسی های http تهیه کنید و یه فایل به اسم proxies.txt توی فولدر اصلی بسازید
سورس کد به صورت خودکار ازش استفاده میکنه
